# 🏥 Ma Santé ++ - Plateforme Santé Complète

## 📋 Aperçu

**Ma Santé ++** est une application healthcare moderne construite avec **React**, **Express.js**, et **SQLite**. Elle permet aux utilisateurs de gérer leurs données médicales de manière sécurisée.

### ✨ Fonctionnalités

- 🔐 **Authentification sécurisée** (JWT + Bcrypt)
- 📅 **Gestion des rendez-vous** médicaux
- 📄 **Gestion des documents** médicaux
- 📊 **Suivi des constantes vitales**
- 👤 **Profil utilisateur** avec infos médicales
- 📱 **Responsive design** (mobile, tablet, desktop)

---

## 🚀 Démarrage Rapide

### Installation & Lancement (One Command)

```bash
# Linux/Mac:
bash start.sh

# Windows:
start.bat

# Puis ouvrir: http://localhost:5173
```
